# Retro Arcade — Modern Retro Website

This is a small, static, responsive site showcasing retro games with a modern visual style.

How to use

- Open `index.html` in your browser.
- Use the search box to filter featured games.
- Click "Toggle Neon" to switch a neon accent theme.

Files

- `index.html` — page structure and content.
- `styles.css` — responsive modern-retro styles.
- `app.js` — small interactions (search and theme toggle).

No build tools required — it's plain HTML/CSS/JS.
