Console Dossier

PlayStation (PS1)

Sony | Launched December 3, 1994

PS1 pushed 3D into the mainstream and opened the door for cinematic game direction using CD storage and memory card progression.

  • Generation: Fifth
  • Media: CD-ROM
  • Controllers: Digital + DualShock

Technical Snapshot

CPU
MIPS R3000A @ 33.9 MHz
Main RAM
2 MB
Video RAM
1 MB
Audio
24-channel ADPCM
Output
Composite / RGB / S-Video

Must-Play Set

  1. Metal Gear Solid
  2. Final Fantasy VII
  3. Resident Evil 2
  4. Gran Turismo 2
  5. Castlevania: Symphony of the Night

Why It Hit Hard

The PS1 shifted game development economics. CD production lowered manufacturing friction, which helped experimental genres and cinematic franchises scale quickly across global markets.

Collector Notes

  • Verify disc drive reads both silver and black-disc games.
  • Inspect controller ports and memory card slot tension.
  • Early model shells can crack near hinge points if over-tightened.
