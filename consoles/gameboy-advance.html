Console Dossier

Game Boy Advance

Nintendo | Launched March 21, 2001

Game Boy Advance gave handheld players stronger processing, a wider display, and one of the most complete portable game libraries ever released.

  • Category: Handheld
  • Media: Cartridge
  • Strength: Deep portable library

Technical Snapshot

CPU
ARM7TDMI @ 16.78 MHz
Aux CPU
8-bit core for compatibility
Screen
240 x 160 reflective LCD
Memory
256 KB external WRAM
Power
2 x AA batteries (original model)

Must-Play Set

  1. Metroid Fusion
  2. Advance Wars
  3. Pokemon Emerald
  4. Castlevania: Aria of Sorrow
  5. Mario and Luigi: Superstar Saga

Why It Hit Hard

GBA carried full-scale design depth into portable sessions. Strategy, action, and RPG genres all found high-quality handheld versions without feeling simplified.

Collector Notes

  • Original screens are dim; modded backlit units vary in quality.
  • Battery door tabs commonly break, especially on heavily used shells.
  • Shoulder buttons can become inconsistent and may need cleaning.
